🚨 Incident Management System (DevOps + AI Ready)
A production-ready Incident Management System built using Flask, MariaDB, Docker, and Docker Compose, designed to simulate real-world SRE workflows.

---

🔧 Tech Stack
Backend: Flask (Python)
Database: MariaDB
Containerization: Docker
Orchestration (Local): Docker Compose
Version Control: Git & GitHub

---

🚀 Features
Create, update, delete incidents
REST API for incident management
Persistent database using MariaDB
Fully containerized application
Multi-container setup using Docker Compose

---

🐳 Run Using Docker Compose
docker compose up --build


Access app at:
👉 http://localhost:5000/

---

📦 Architecture
Flask app runs in one container
MariaDB runs in another container
Both connected via Docker network

---

🎯 Why MariaDB?
Better scalability than JSON
Supports concurrent users
Structured data storage

---

📈 Future Enhancements
Kubernetes deployment
CI/CD using Jenkins
AI-based incident analysis
Monitoring using Prometheus & Grafana
